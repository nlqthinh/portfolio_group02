Website Find Hostel

Technology used: HTML, CSS, JS, MySQL, Wordpress, PHP.

Main plugins in the project: Advanced Custom Fields, WP Job Manager, GeoDirectory, Custom Post Type UI.

Position in the project: Full-stack developer.

Number of participants: 10.

Status: Built and developed a website to find quality hostels, meeting the requirements and high standards of security, comfort and living environment.

Description: The Find Hostel website helps students find the best hostel for students in the fastest way and fits their budget.

User Interface Design

An intuitive and aesthetically appealing user interface was the top consideration for this project. Using HTML, CSS, and JavaScript, I have meticulously created a visually appealing interface that provides a smooth and engaging user experience.

Find hostels on google map

To allow students to search for hostels based on location, I have integrated the Google Maps API into the website. Leveraging JavaScript and PHP, I have developed a dynamic search feature that is integrated with the WP Job Manager plugins. and GeoDirectory. This allows students to visualize hostels on a map, filter results based on specific areas, and explore nearby amenities. The seamless integration of Google Maps has enriched the search experience, providing students with an interactive and comprehensive way to find the right hostels.

Post to the forum

Realizing the importance of community engagement, I incorporated the posting functionality into the website. Leveraging the Custom Post Type UI and Advanced Custom Fields plugins, I have developed a secure and user-friendly system for students and property owners to contribute related articles, insights, and advice. regarding student housing. This feature fostered a sense of community and provided a valuable resource for users looking for information and guidance.
